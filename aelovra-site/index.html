About the Studio

One developer.
One obsession.

Aelovra is an indie software studio — just one developer building clean, focused tools for developers and makers. The name has no meaning; it was made up to sound distinct and memorable.

The goal is simple: a small suite of quality indie tools, built with care and funded by a Patreon Premium tier so development never stops. No bloat. No dark patterns. Just software that does exactly what it says.

Right now we’re shipping MiniTools Hub — the flagship app. More tools in the suite are on the way.

MiniTools Hub
